Sinqobile Construction Logo

📸 Instructions to Create PNG Version:

  1. Open this HTML file in your browser (or view the SVG directly)
  2. Take a screenshot of the logo at exactly 512x512px, OR
  3. Use an online SVG to PNG converter like:
    • cloudconvert.com
    • svgtopng.com
  4. Set output dimensions to 512x512px
  5. Save the PNG file as public/logo.png

Alternative: Use ImageMagick command:
magick convert -background none -size 512x512 logo.svg logo.png
